## How to Run
1. Clone the repository and copy the `.env.example` to `.env`
2. Add your `DISCORD_TOKEN` to the `.env`
3. Start the bot using `cargo run`

## Languages

### Supported:
- Polish
- French
- German
- Spanish
- Romanian
- Turkish
- Dutch
- Swedish
- Slovenian

### Unsupported:
- Arabic
- Chinese
- Persian
- Korean
- Japanese
